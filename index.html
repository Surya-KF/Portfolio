AUTOMATOS

AUTOMATOS is an advanced automated surveillance and messaging system designed to enhance security and monitoring capabilities using video footage. Built with precision and scalability in mind, this project leverages Python and state-of-the-art YOLO models (YOLOv5 and YOLOv8) for real-time detection and identification of individuals and activities. AUTOMATOS is tailored to meet the needs of organizations seeking to bolster their security infrastructure, streamline monitoring processes, and improve overall situational awareness. MediSaga

MediSaga is an intelligent medical question-answering system that combines Retrieval-Augmented Generation (RAG) with advanced indexing techniques using RAPTOR and Milvus. Designed to deliver accurate and contextually relevant answers to medical inquiries, MediSaga harnesses the power of Llama2 to ensure high-quality responses. The project features robust PDF text extraction and query expansion capabilities, offering a seamless and intuitive user experience through a Streamlit-based UI. Whether you are a healthcare professional seeking precise information or an individual with medical questions, MediSaga provides a reliable and efficient solution for acquiring accurate medical knowledge with ease.

View Project

Churn Prediction for Telecom

The Churn Prediction project is a machine learning-based solution focused on identifying and reducing customer churn in the telecom sector. By analyzing customer behavior and transaction data, this project predicts the likelihood of customers leaving the service, enabling telecom companies to take proactive measures to retain them. The model employs advanced data processing and classification techniques to generate actionable insights, helping providers improve customer retention rates and minimize revenue losses.
